# Duedle
Due + Moodle = Duedle
## 免責事項
この拡張機能の使用により生じたいかなる損害についても、開発者は一切の責任を負いません。
## Duedleって何？
名工大のMoodleページをより快適に利用できるようにパワーアップする拡張機能です。
## Duedleのメイン機能
- ダークモード
- わかりやすい課題一覧をトップに表示
- コース表示順のカスタマイズ
- コースページで折りたたんだ項目の表示/非表示
- 冗長な情報の削減
- ページの閲覧履歴を左に表示
- ファイルをブラウザ上で閲覧

[詳細はこちら](guideline.md)
